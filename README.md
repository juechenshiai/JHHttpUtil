# JHHttpUtil

#### 项目介绍
OkHttp3 封装工具类  
基于OkHttp3进行封装，实现GET/POST/上传/下载的快捷操作  
另外简单封装了线程工具，单线程/多线程/UI线程

#### 使用说明

1. 入口 JH类
2. 线程使用 `JH.thread()`
3. http请求 `JH.http()`
